What is CSS?

  1. CSS stands for Cascading Style Sheets.
  2. CSS describes how html elements are to be displayed on screen, paper, or in other media.
  3. Css is not neccesary CSS saves a lot of work. It an control the layout of multiple webpages all at once.
  4. External stylesheets are stored in CSS files.

CSS can be added to HTML documents in 3 ways:

  • Inline-by using the style attribute inside HTML elements1
  • Internal-by using a style element in the head section
  • External-by using a link element to link an external CSS file.
  • The most common way to add CSS. is to keep the styles in external CSS files. However ,in this tutorial we will use inline and internlstyles.because this easier to demonstrate, and easier for you to try it yourself.

When comparing CSS class vs ID, the difference is that CSS class applies to multiple elements ID, on the other hand applies a style to one unique element.ID is also special in that you can use a speial URL to link directly to an element and it's used by JavaScript.

In CSS selectors are used tp target a specific element or range of elements on a web page.Once an element has been targeted, a style or set of styles can be applied to the element.

There is a wide range of selection available.Two of the most coomonly used are class and ID.Both are used to target elements to which a style should be applied.
